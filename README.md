# MBT and Aggression Behavioral Analysis

This project explores behavioral changes before and after MBT intervention using Python-based behavioral data analysis.

## Overview

The notebook examines behavioral variables including:
- Physical aggression
- Verbal aggression
- Relational aggression
- General aggression

The project uses descriptive statistics and data visualization to compare behavioral outcomes between groups.

## Tools Used

- Python
- Pandas
- Matplotlib
